QUICK SETUP GUIDE

New to Protocol Seven? Set up in 5 steps:

1. SETTINGS TAB → Set your character name and avatar
2. MISSION TYPE → Select your pillar (Body, Mind, Money, etc.)
3. CHOOSE MISSION → Pick from 32+ missions or create custom
4. SET GOAL & DEADLINE → Define your target and timeline
5. START TRACKING → Come back here to log progress daily
📊 TRACKING TIP: You don't need to log whole numbers. Use decimals like 0.1, 0.2, or 0.5 for partial progress. The goal is truthfulness, not perfection. Track what's real.
⚠️ IMPORTANT: Go to SETTINGS → Data Management → EXPORT SAVED GAME to backup your progress before closing. Your data saves automatically, but always export before major changes!
📱 INSTALL AS APP: Add Protocol Seven to your home screen for quick access!
iPhone: Tap Share → "Add to Home Screen"
Android: Tap Menu → "Add to Home screen"
Desktop: Click install icon in address bar
Pro Tip: Each mission saves its own progress automatically. Switch between missions anytime in Settings to track multiple goals.

XP VALUE GUIDE
Use this guide to set appropriate XP values:
20-30 XP: Quick Routines
⏱ 5-10 minutes | Examples: Making bed, hygiene
40-60 XP: Foundation Habits
⏱ 15-30 minutes | Examples: Meditation, exercise
80-100 XP: High-Impact Work
⏱ 60-90 minutes | Examples: Deep work, outreach
120-150 XP: Critical Operations
⏱ 90+ minutes | Examples: Major projects
⚖️ BALANCE PRINCIPLE:
Total daily XP should be 1000-1500. Set values based on TIME + IMPACT.
